# 💰 Expense Tracker Web Application

A simple and beginner-friendly Expense Tracker Web Application built using Python Django.  
This project helps users manage and track daily expenses with authentication and dashboard features.

---

## ✨ Features

- User Registration & Login
- Add, Edit & Delete Expenses
- Expense Categories
- Dashboard Analytics
- Monthly Expense Tracking
- Responsive Bootstrap UI
- Django Admin Panel

---

## 🛠 Tech Stack

- Python Django
- SQLite
- HTML
- CSS
- Bootstrap 5

---

## 🚀 Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
